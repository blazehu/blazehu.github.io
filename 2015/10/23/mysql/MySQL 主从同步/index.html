MySQL 主从同步

2015-10-23 DB MySQL 评论

简介

主从同步分三个步骤：

  1. master 将改变记录到二进制日志 binary log 中（这些记录叫做二进制日志事件，binary log events）。
  2. slave 将 master 的 binary log events 拷贝到它的中继日志 relay log 中。
  3. slave 重做中继日志中的事件，将改变反映它自己的数据。
