Rundeck 简介

2020-03-20 运维 评论

Rundeck 是一个基于 Java 和 Grails 的开源的运维自动化工具，提供了 Web 管理界面进行操作，同时提供命令行工具和 WebAPI 的访问控制方式。Rundeck 能够帮助开发和运维人员更好地管理各个节点。

安装

安装方式比较多，这里使用 war 包来简单介绍安装的步骤。
